
# Netflix Content Analysis | Python EDA Project

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a Netflix dataset containing over 8,400+ records. The goal is to analyze content trends across genres, ratings, countries, and release years using Python.

---

## 🧰 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Key Objectives
- Clean and preprocess Netflix dataset
- Handle missing values and inconsistent data
- Analyze content distribution across different categories
- Identify trends in Netflix content over time

---

## 📈 Analysis Performed
- Univariate analysis (ratings, genres, etc.)
- Bivariate analysis (type vs year, country vs content type)
- Multivariate analysis for deeper insights

---

## 📌 Key Insights
- Most content belongs to Movies category
- Certain countries dominate Netflix content production
- Content addition increased significantly after 2015

---

## 📂 Dataset
Netflix Movies and TV Shows Dataset (Kaggle)

---

## 🚀 How to Run
1. Clone the repository
2. Open notebook in Google Colab or Jupyter Notebook
3. Install required libraries
4. Run all cells
